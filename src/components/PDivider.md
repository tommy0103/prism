# PDivider

Horizontal rule with an optional centered label.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | Optional text displayed centered on the divider line |

## Example

```html
<p-divider label="分隔标签"></p-divider>
<p-divider></p-divider>  <!-- plain hr -->
```

## Notes

- Without a `label`, renders as a plain horizontal rule.
- Use between major sections for visual breathing room.
